Cave Game 2D

This was a course project designed to teach us about UI development from a programming-heavy perspective. The game also featured seed-based terrain generation, ensuring no two levels were the same - I fondly look back on being locked into a library study room for hours on end trying to figure out how to implement enemies that chased and attacked the player. Definitely the most fun project I've done, that taught me nearly everything I know about Python as well as Object Oriented Programming.
